📊 Data Flow Through Layers
1
User Interaction (Frontend)
ผู้ใช้คลิกปุ่ม, เลือก Filter, หรือไปยังหน้าอื่น → Frontend capture event
2
State Update (Frontend)
Context State เปลี่ยน → Components re-render ด้วยข้อมูลใหม่
3
API Request (Frontend → Backend)
Frontend ส่ง HTTP Request ไปยัง Backend API เพื่อขอข้อมูล
4
Backend Processing
Backend ประมวลผล Request → Query Database → Calculate Metrics
5
Database Query (Backend → Database)
Backend ส่ง SQL Query ไปยัง Database เพื่อดึงข้อมูล
6
Data Response (Database → Backend)
Database ส่ง Data กลับมายัง Backend → Backend แปลงเป็น JSON
7
API Response (Backend → Frontend)
Backend ส่ง JSON Response กลับมายัง Frontend
8
UI Render (Frontend)
Frontend update State → Components re-render → Display ข้อมูลใหม่บนหน้าจอ
