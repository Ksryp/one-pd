🎯 Recommendation for SNK MES:

ใช้ 99% Uptime เพราะว่า:
• Dashboard นี้ใช้ในการ monitor production (critical system)
• ลงนาน 3 ชั่วโมงต่อปี vs 18 ชั่วโมง = ไม่มากที่สอง
• ค่าเพิ่ม $60-80/ปี = ประหยัดมาก เมื่อเทียบกับ benefit
• ถ้า production หยุด = สูญเสีย production time นาน

Cost difference: เพิ่มแค่ $5-6/เดือน = ~$70/ปี สำหรับ 99% uptime
💡 Why 99% costs more?

• ต้อง 24/7 Monitoring (CloudWatch) → +$5-10
• ต้อง Auto-scaling group (standby instance) → +$15-20
• ต้อง Alerting system (SNS) → +$1-5
• But: 95% คุณต้อง manual monitoring = มีคน on-call
• จริง ๆ แล้ว automatic 99% ถูกกว่า manual 95%!
