SNK MES Dashboard

Complete Data Flow & Layer Connections

1

User Interaction

Component: Dashboard.jsx
Action: User clicks button or selects filter
Result: Event triggered (onClick, onChange)
2

State Update

Component: DashboardContext
Action: setSelectedDate('today')
Result: All components re-render with new state
3

Routing Decision

Tool: React Router v6
Action: navigate('/stage/slip-prep')
Result: URL changes to /stage/slip-prep
4

Fetch Data

Function: useEffect hook
Action: fetch('/api/stage/slip-prep?date=today')
Result: HTTP GET request sent to Backend
5

Backend Processing

Endpoint: GET /api/stage/:stageId
Process: Validate token → Check permissions
Next: Calculate metrics and query database
6

Business Logic

Calculate: OEE = Good / (Good + Scrap + Repair)
Calculate: Takt = Total Time / Units Produced
Process: Aggregate defect data
7

Database Query

Query: SELECT * FROM production_data
Filter: WHERE stageId='slip-prep' AND date='2026-05-19'
Result: Data fetched from database
8

Response Ready

Format: JSON with all calculated metrics
Status: 200 OK
Next: Send back to Frontend
1

Database Returns

What: Raw data rows from PostgreSQL/InfluxDB
Data: production_data, stage_metrics, defect_log tables
Time: Usually < 100ms
2

Backend Formats

Action: Serialize to JSON format
Add: Calculated fields (OEE, Takt)
Set: HTTP headers (Content-Type, CORS)
3

Send HTTP 200 OK

Status: 200 (Success)
Body: Complete JSON response
Time: < 50ms network latency
4

Frontend Receives

Event: fetch promise resolves
Parse: response.json() → JavaScript object
Save: Store in component state
5

Update State

Action: setStageData(response)
Effect: DashboardContext updates
Trigger: All subscribed components re-render
6

UI Render

Components: OEECard, TaktCard, StageChart
Data: Bind to props from new state
Display: User sees updated Dashboard
Frontend State Structure
interface DashboardContextType { theme: 'light' | 'dark'; selectedDate: 'today' | 'yesterday' | string; selectedDefect: 'all' | 'pinhole' | 'crack' | 'discoloration'; selectedParameter: string[]; selectedModel: string; alerts: Alert[]; stageData: StageDetail | null; }
API Request Example
const fetchStageData = async (stageId: string) => { const response = await fetch( `/api/stage/${stageId}?date=${selectedDate}`, { method: 'GET', headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${token}` } } ); const data = await response.json(); setStageData(data); };

Data Flow Steps Summary
Complete Flow:

User Action → State Update → useEffect Triggers → fetch() Request → Backend Route Handler → Business Logic → Database Query → Format Response → HTTP 200 OK → Frontend Receives → Parse JSON → setState() → Components Re-render → User Sees New Data
