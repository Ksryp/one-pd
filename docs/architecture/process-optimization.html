💡 การปรับ Process — เรียงจากผลลัพธ์ดีที่สุด

⚡ High Impact เติม Deflocculant / ปรับลด Viscosity โดยตรง ▼
ปัญหาที่แก้
  • Viscosity V0 = 940 cP เกิน UCL 900 cP (+4.4%)
  • Slip หนืดเกินไป → Air bubble ติดค้าง → Pinhole
  • Flow rate ในแม่พิมพ์ต่ำ → Casting rate ลด
วิธีการปรับ
  1. วัด Viscosity V0 ซ้ำด้วย Viscometer → ยืนยันค่า 940 cP
  2. เติม Sodium Silicate (Deflocculant) ทีละ 0.05% → คนผสม 5 นาที → วัดซ้ำ
  3. เป้าหมาย: ลด Viscosity V0 → 820–860 cP (ใน Control Zone)
  4. ถ้า Viscosity ยังสูง → เติมน้ำ DI 0.5–1% ของ batch weight
  5. Recheck V30 ให้อยู่ใน 680–880 cP พร้อมกัน

⚡ High Impact ตรวจสอบ Ball Mill Speed + อุณหภูมิน้ำ (Root Cause) ▼
ปัญหาที่แก้
  • Viscosity สูงซ้ำๆ ในช่วงเวลาเดิม → ต้นเหตุไม่ได้แก้
  • Ball Mill หมุนเร็วเกินไป → Particle ละเอียดเกิน → Viscosity สูง
  • น้ำอุ่นเกินไป → Deflocculant เสื่อมประสิทธิภาพ
วิธีการปรับ
  1. ตรวจ Ball Mill RPM → เทียบกับ Recipe spec → ปรับลดถ้าสูงเกิน 5%
  2. วัดอุณหภูมิน้ำที่ใช้ผสม → ต้องอยู่ที่ 20–26°C (ห้ามเกิน 28°C)
  3. ตรวจสอบ Grinding time → ถ้านานเกิน → ลด 10–15 นาที
  4. ตรวจ Particle Size Distribution → ควร D50 < 8 μm
  5. Log ผลทุก batch → หา Pattern ว่าเกิดช่วงกะไหน

💡 การปรับ Process — เพื่อป้องกัน & เพิ่มคุณภาพ

⚡ High Impact ลด Particle Size → ป้องกัน Pinhole ใน Glaze ▼

  • Particle ใหญ่เกิน (>3.5%) → ละลายไม่หมดใน Kiln → Pinhole บนผิว Glaze

  1. เพิ่มเวลา Milling 15–20 นาที → ตรวจ Particle Size ใหม่ด้วย Laser Diffraction
  2. Target: D50 < 6 μm, % Residue < 0.6% (ร่อนตะแกรง 45 μm)
  3. เปลี่ยน Grinding Media ถ้าสึกเกิน 20% ของน้ำหนักเริ่มต้น

💡 การปรับ Process — เรียงจากผลลัพธ์ดีที่สุด

⚡ High Impact เปลี่ยน / Repair Mold ที่สึกหรอ ▼
ปัญหาที่แก้
  • Mold เก่า ดูดซับน้ำจาก Slip ได้ช้า → Cycle time ยาว
  • ผิว Mold หยาบ → Grain Size ชิ้นงานใหญ่
  • Mold ร้าว/แตก → Seam line บนชิ้นงาน, Deformation
วิธีการปรับ
  1. Inspect Mold ทุกใบ → วัด Mold Cycle แยกรายใบ → หาตัวที่ช้าผิดปกติ (>33 min)
  2. Mold ที่ Cycle >33 min → ส่ง Repair (อุดรอย, ขัดผิว) หรือ Replace
  3. ตรวจสอบ Mold Age → หมุนเวียน Mold ตามอายุ (เช่น ไม่เกิน 500 cycles)
  4. ทาน้ำยา Mold Release ใหม่หลัง Repair ทุกครั้ง

⚡ High Impact ปรับ Slip Viscosity → ลด Casting Time ▼
ปัญหาที่แก้
  • Viscosity V0 สูง (940 cP) → Slip ไหลเข้า Mold ช้า → Wall formation ช้า

  2. ทดสอบ Casting time หลังปรับ Viscosity → ควรลดได้ 6–10 นาที
  3. อย่าให้ Viscosity ต่ำเกินไป (<700 cP) → Slip บางเกิน → Wall얇 เกิน → Crack

⚡ High Impact ปรับ Drying Curve — ลด Moisture ให้ต่ำ & สม่ำเสมอ ▼
ปัญหาที่แก้
  • Moisture 13.2% ใกล้ขอบ UCL → risk สูงถ้า Casting มาในช่วง Mold Cycle ยาว
  • Drying ไม่สม่ำเสมอ (Core ชื้นกว่า Surface) → Crack ใน Kiln

  3. ติดตั้ง Moisture Sensor หลายจุดในรถเข็น → Map ความสม่ำเสมอ
  4. หมุน/พลิกชิ้นงานระหว่าง Drying ถ้าทำได้ → Uniform drying

💡 การปรับ Process — เรียงจากผลลัพธ์ดีที่สุด

⚡ High Impact เร่ง Maintenance Kiln No.2 ให้กลับมา NORMAL ▼
ปัญหาที่แก้
  • Cooling Zone temperature ต่ำ → Thermal gradient สูง → Crack เพิ่มมาก
  • Kiln ไม่ได้ Full capacity → WIP คั่ง → OEE ลด
  • Uneven temperature distribution → Discoloration บางตำแหน่งใน Car
วิธีการปรับ
  1. Prioritize: ตรวจ Cooling Zone Burner / Fan → ซ่อมภายใน 4–8 ชั่วโมง
  2. ตรวจ Thermocouple ใน Cooling zone → ถ้า Drift > 20°C → เปลี่ยนทันที
  3. ทำ Temperature profile run (Empty car) ก่อน Resume production
  4. ตรวจ Kiln Car refractory → อุดรอยร้าวด้วย Ceramic fibre
ผลลัพธ์ที่คาดหวัง
Firing Yield↑ จาก 81% → 86–88%
Crack ใน Firing↓ 55–65%
Discoloration↓ 40–50%
OEE↑ 8–12%
ประสิทธิผลโดยรวม: ~90%
⚡ High Impact ปรับ Temperature Ramp Rate ใน Critical Zone ▼
ปัญหาที่แก้
  • Ramp rate เร็วเกินไปใน 573°C (Quartz inversion) → Crack
  • Ramp เร็วที่ 300–400°C → Burn-out organic too fast → Bloating/Crack
วิธีการปรับ
  1. ช่วง 500–620°C: ลด Ramp rate → ไม่เกิน 80°C/hr (จากเดิมอาจ 120°C/hr)
  2. ช่วง 300–450°C: ลด Ramp rate → ไม่เกิน 60°C/hr (Binder burn-out zone)
  3. ช่วง Cooling 600–500°C: ลด Cooling rate → ไม่เกิน 100°C/hr
  4. ทดสอบ: Run Profile ใหม่กับ 1 Car → ตรวจ Crack rate ก่อน Full production
ผลลัพธ์ที่คาดหวัง
Crack จาก Thermal shock↓ 45–60%
Firing Cycle↑ +1–1.5 hr
Firing Yield↑ 3–5%
ประสิทธิผลโดยรวม: ~75%
🔶 Medium Impact ปรับ Loading Density & Car Arrangement ▼
ปัญหาที่แก้
  • โหลดชิ้นงานแน่นเกินไป → ลม circulation ไม่ดี → Hot/Cold spot
  • ชิ้นงานกันลม → ชิ้นด้านในได้รับความร้อนน้อย → Discoloration
วิธีการปรับ
  1. ลด Loading density 10–15% → เพิ่มระยะห่างระหว่างชิ้นงาน 5–8 mm
  2. ใช้ Setters/Spacers ที่ถูกต้อง → ป้องกันชิ้นงานเอียงหรือแตะกัน
  3. Stagger การวางชิ้นงานในแนวดิ่ง → ให้ลมผ่านได้ทุก Level
ผลลัพธ์ที่คาดหวัง
Discoloration (position-related)↓ 30–40%
Throughput↓ 10–15% ต่อ Car
Net Yield↑ เล็กน้อย (Reject ลด)
ประสิทธิผลโดยรวม: ~48%
🔶 Medium Impact ตรวจสอบ Glaze Fit (Expansion Coefficient) ▼
ปัญหาที่แก้
  • Glaze expansion มาก/น้อยกว่า Body → Crazing หรือ Peeling หลัง Cooling
  • Glaze Fit ไม่ดี → Crack เป็นร่างแหบนผิว
วิธีการปรับ
  1. วัด Thermal Expansion Coefficient (TEC) ของ Body และ Glaze
  2. Target: TEC Glaze ต่ำกว่า TEC Body เล็กน้อย (Compressive stress in glaze)
  3. ถ้า Misfit >0.5×10⁻⁶/°C → ปรับ Formula Glaze เพิ่ม/ลด Silica-Alumina
ผลลัพธ์ที่คาดหวัง
Crazing / Peeling↓ 60–80%
ผลต่อ Defect อื่นน้อย
ประสิทธิผลโดยรวม: ~40% (เฉพาะ Crazing)
